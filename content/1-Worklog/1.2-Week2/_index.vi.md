### Kết quả đạt được tuần 2:

- Hiểu AWS là gì và nắm được các nhóm dịch vụ cơ bản:
  - Compute
  - Storage
  - Networking
  - Database
  - ...

- Đã tạo và cấu hình AWS Free Tier Account thành công.

- Làm quen với AWS Management Console và biết cách tìm kiếm, truy cập các dịch vụ AWS.

- Cài đặt và cấu hình AWS CLI trên máy tính bao gồm:
  - Access Key
  - Secret Access Key
  - Default Region
  - Output Format

- Thực hiện thành công các lệnh AWS CLI cơ bản để kiểm tra tài khoản, cấu hình và quản lý dịch vụ AWS.

- Có khả năng sử dụng kết hợp AWS Management Console và AWS CLI để quản lý tài nguyên AWS, tạo nền tảng cho các nội dung học tập ở các tuần tiếp theo.
